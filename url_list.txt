说明：
	此文件格式：
		链接 ：链接描述
			get：数据内容（简单字段，字典类型数据用括号包裹，数组用花括号包裹）
			post：数据内容
请使用空格作为间隔符。

url(后端api接口):
	/api/account/login : login page
		get : 请求参数为空即可，若已登录，返回code=0，否则返回code!=0
		post : 请求参数为{username, password}，登录成功返回code=0，否则返回code!=0
	/api/account/logout : logout page
		post : 请求参数为空，登出成功返回code=0，否则返回code!=0
	/api/account/detail/room_list : roomlist detail
		get : 请求参数为空，返回数据格式
			{
				user(username, room_num),
				roomlist{(num, title, description, state, interviewer_name}, 
			}
		post : 修改房间信息，请求参数： {roomnum, title, description,interviewer_email}, 编辑成功返回code=0，否则返回code!=0
	/api/account/detail/room_list/invite_interviewer : 邀请面试官
		post : 请求参数： {email}, 邮件发送成功返回code=0，否则返回code!=0
	/api/account/detail/room_list/close : close the room
		post : 请求参数： {roomnum}, 关闭成功返回code=0，否则返回code!=0
	/api/accout/detail/interviewee_list : 候选人列表
		get : 请求参数为空，返回数据格式
			interviewee_list{(num, username, email, telephone, address, state)}
		post : 修改候选人信息，请求参数: {num, username, email, telephone, address}
		编辑成功返回code=0，否则返回code!=0
	/api/accout/detail/interviewee_list/delete : 
		post : 删除候选人，请求参数：{num}
		删除成功返回code=0，否则返回code!=0
	/api/accout/detail/interviewee_list/add :
		post : 添加候选人，请求参数：{username, email, telephone, address}
		删除成功返回code=0, num，否则返回code!=0
	/api/account/detail/interviewee_list/sendemail :
	    post : 给候选人发送面试房间链接的邮件，请求参数：{userid, roomid}
	    发送成功返回code=0，否则返回code=1
